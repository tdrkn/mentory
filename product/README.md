# ER-диаграммы продукта

В папке оставлены две рабочие ER-диаграммы:

- `er-user-panel.puml` — схема пользовательской части.
  Здесь показан основной путь: пользователь, профиль, услуги, слоты, сессии, чат, оплата и отзыв.
  Нужна, чтобы быстро понимать, как работает продукт для mentor/mentee.

- `er-admin-panel.puml` — схема админской части (Trust).
  Здесь показаны жалобы, сообщения по жалобам, файлы, проверка документов ментора и действия модерации.
  Нужна, чтобы понимать, как админ обрабатывает спорные ситуации и контроль качества.

Почему две схемы отдельно:

- Так проще читать и обсуждать.
- Пользовательские процессы и админские процессы не смешиваются в одну перегруженную картинку.
